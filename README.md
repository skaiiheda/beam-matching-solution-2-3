# Панель управления согласованием пучка

### Возможности

- **Интерактивная панель параметров**: Настройка параметров Твисса, конфигурации пучкового канала и сил квадруполей
- **Визуализация в реальном времени**: Все графики обновляются мгновенно при изменении параметров
- **Оптимизация согласования пучка**: Алгоритм Нелдера-Мида для автоматического поиска оптимальных настроек квадруполей
- **Множественные визуализации**:
  - Схематическая диаграмма пучкового канала (на основе Plotly)
  - β-функции вдоль пучкового канала
  - Эллипсы фазового пространства (плоскости X и Y с вкладками)
  - Графики огибающей пучка (σ = √(β·ε))
  - Таблица статистики с расчётом ошибок
- **Физические вычисления**: Полная библиотека оптики пучка с распространением Твисса
- **Одностраничное приложение**: На основе Streamlit с адаптивной компоновкой

### Установка

1. **Перейдите в каталог проекта**:
```bash
cd py-implementation
```

2. **Создайте и активируйте виртуальное окружение** (если ещё не создано):
```bash
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

3. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

### Запуск панели управления

Запустите приложение Streamlit:

```bash
streamlit run app.py
```

Или используйте виртуальное окружение:
```bash
./venv/bin/streamlit run app.py
```

Панель управления откроется в вашем браузере по адресу `http://localhost:8501`

### Руководство по использованию

#### Панель параметров (Боковая панель)

**Входные параметры Твисса**
- βx, αx: Бета и альфа в плоскости X на входе пучка
- βy, αy: Бета и альфа в плоскости Y на входе пучка

**Целевые параметры Твисса**
- βx*, αx*: Желаемые значения в плоскости X на выходе пучка
- βy*, αy*: Желаемые значения в плоскости Y на выходе пучка

**Конфигурация пучкового канала**
- Длина дрейфа: Расстояние между квадруполями (0.5-2.0 м)
- εx: Эмиттанс в плоскости X (нм·рад)
- εy: Эмиттанс в плоскости Y (нм·рад)

**Силы квадруполей**
- k₁, k₂, k₃, k₄: Силы фокусировки/дефокусировки квадруполей (м⁻²)
  - Положительное k: Фокусировка в плоскости X, дефокусировка в плоскости Y
  - Отрицательное k: Дефокусировка в плоскости X, фокусировка в плоскости Y

**Кнопки действий**
- 🔄 Сброс: Сбросить все параметры к значениям по умолчанию
- ▶ Оптимизировать: Запустить оптимизацию Нелдера-Мида для согласования целевого Твисса

#### Визуализации

**Компоновка пучкового канала**
- Схематический вид Q1 → Дрейф → Q2 → Дрейф → Q3 → Дрейф → Q4
- Цветовая кодировка: Зелёный (фокусировка), Красный (дефокусировка)
- Отображение значений k для каждого квадруполя

**Статистика согласования**
- Таблица сравнения входных, выходных и целевых значений
- Процентные ошибки для каждого параметра
- Статус согласованности/несогласованности на основе средней ошибки
- Отображение максимальной ошибки

**β-функции вдоль пучкового канала**
- Кривые βx и βy вдоль пучкового канала
- Линии отсчёта на позициях квадруполей
- Линии отсчёта целевого β* (пунктирные)
- Интерактивное масштабирование/перемещение

**Фазовое пространство**
- Вкладки для плоскости X и плоскости Y
- Три эллипса для каждой плоскости:
  - Входной (пунктирный оранжевый)
  - Выходной (сплошной розовый)
  - Целевой (точечный зелёный)
- Всплывающие подсказки с информацией о позиции

**Огибающая пучка**
- Диаграммы с заполненными областями, показывающие σ = √(β·ε)
- Огибающие X и Y
- Маркеры позиций квадруполей
- Отображение эмиттанса

### Реализация физики

**Матрицы переноса:**
- Дрейф: `M = [[1, L], [0, 1]]`
- Квадруполь (тонкая линза): `M = [[1, 0], [-k, 1]]`

**Распространение Твисса:**
```
β₂ = m₁₁²β₁ - 2m₁₁m₁₂α₁ + m₁₂²γ₁
α₂ = -m₁₁m₂₁β₁ + (m₁₁m₂₂ + m₁₂m₂₁)α₁ - m₁₂m₂₂γ₁
γ = (1 + α²) / β
```

**Эллипс фазового пространства:**
```
x = √ε·√β·cos(θ)
x' = √ε·(-α/√β·cos(θ) + 1/√β·sin(θ))
```

**Огибающая пучка:**
```
σ = √(β·ε)
```

**Ошибка согласования:**
```
Error = (βx_out - βx_target)² + (αx_out - αx_target)²
       + (βy_out - βy_target)² + (αy_out - αy_target)²
```

**Оптимизация:**
- Алгоритм: Нелдер-Мид (симплекс-метод)
- Цель: Минимизировать ошибку согласования
- Переменные: k₁, k₂, k₃, k₄
- Ограничения: Без ограничений (-∞, ∞)
