# Панель управления согласованием пучка

### Возможности

- **Интерактивная панель параметров**: Настройка параметров Твисса, конфигурации пучкового канала и сил квадруполей
- **Визуализация в реальном времени**: Все графики обновляются мгновенно при изменении параметров
- **Оптимизация согласования пучка**: Двухэтапный алгоритм (differential_evolution + Nelder-Mead) для автоматического поиска оптимальных настроек квадруполей
- **Поддержка 4 или 5 квадруполей**: Переключение между стандартной FODO-ячейкой и расширенной конфигурацией
- **Штраф за большие β**: Ограничение β-функций во время оптимизации для предотвращения расходимости
- **Множественные визуализации**:
  - Схематическая диаграмма пучкового канала (на основе Plotly)
  - β- и α-функции вдоль пучкового канала
  - Эллипсы фазового пространства (плоскости X и Y с вкладками)
  - Графики огибающей пучка (σ = √(ε·β))
  - Таблица статистики с расчётом ошибок
- **Физические вычисления**: Полная библиотека оптики пучка с распространением Твисса и толстыми квадруполями

### Установка

1. **Перейдите в каталог проекта**:
```bash
cd py-implementation
```

2. **Создайте и активируйте виртуальное окружение**:
```bash
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac
```

3. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

### Запуск панели управления

```bash
streamlit run app.py
```

Панель управления откроется в браузере по адресу `http://localhost:8501`

### Руководство по использованию

#### Панель параметров (Боковая панель)

**Конфигурация пучкового канала**
- Число квадруполей: 4 или 5
- εx, εy: Эмиттанс в плоскостях X и Y (нм·рад)
- Длина квадруполя: Физическая длина каждого квадруполя (м)

**Входные параметры Твисса**
- βx, αx: Бета и альфа в плоскости X на входе
- βy, αy: Бета и альфа в плоскости Y на входе

**Целевые параметры Твисса**
- βx*, αx*: Желаемые значения в плоскости X на выходе
- βy*, αy*: Желаемые значения в плоскости Y на выходе

**Параметры штрафа за большие β**
- Использовать штраф: Включение/выключение ограничения β
- β_limit: Максимальное допустимое значение β (м)
- Вес штрафа: Сила штрафной функции

**Силы квадруполей**
- k₁, k₂, k₃, k₄: Силы фокусировки/дефокусировки (м⁻²)
- k₅: Дополнительный квадруполь (только при выборе 5 линз)
  - Положительное k: Фокусировка в X, дефокусировка в Y
  - Отрицательное k: Дефокусировка в X, фокусировка в Y

**Кнопки действий**
- 🔄 Сброс: Сбросить все параметры к значениям по умолчанию
- ▶ Оптимизировать: Запустить двухэтапную оптимизацию

#### Визуализации

**Компоновка пучкового канала**
- Схематический вид квадруполей и дрейфов
- Цветовая кодировка: Зелёный (фокусировка), Красный (дефокусировка)
- Отображение значений k для каждого квадруполя

**Статистика согласования**
- Таблица сравнения входных, выходных и целевых значений
- Процентные ошибки для каждого параметра
- Статус согласованности на основе средней ошибки (<5%)
- Максимальная ошибка

**α- и β-функции**
- Верхний график: βx и βy вдоль пучкового канала
- Нижний график: αx и αy вдоль пучкового канала
- Линии отсчёта на позициях квадруполей
- Пунктирные линии целевых значений

**Фазовое пространство**
- Вкладки для плоскости X и плоскости Y
- Три эллипса: Входной (оранжевый пунктирный), Целевой (зелёный точечный), Текущий (розовый сплошной)
- Выбор позиции: Вход, После Q_i, Выход

**Огибающая пучка**
- Заполненные области для σx и σy (σ = √(β·ε))
- Маркеры позиций квадруполей
- Отображение эмиттанса

### Реализация физики

**Матрицы переноса:**
- Дрейф: `M = [[1, L], [0, 1]]`
- Квадруполь (толстый, k > 0): `M = [[cos(√kL), sin(√kL)/√k], [-√k sin(√kL), cos(√kL)]]`
- Квадруполь (толстый, k < 0): `M = [[cosh(√|k|L), sinh(√|k|L)/√|k|], [√|k| sinh(√|k|L), cosh(√|k|L)]]`

**Распространение Твисса:**
```
β₂ = m₁₁²β₁ - 2m₁₁m₁₂α₁ + m₁₂²γ₁
α₂ = -m₁₁m₂₁β₁ + (m₁₁m₂₂ + m₁₂m₂₁)α₁ - m₁₂m₂₂γ₁
γ = (1 + α²) / β
```

**Ошибка согласования:**
```
Error = Mismatch(x)² + Mismatch(y)²
Mismatch = 0.5 * (β_target·γ_out - 2α_target·α_out + γ_target·β_out) - 1
```

**Оптимизация:**
- Этап 1: differential_evolution (глобальный поиск)
- Этап 2: Nelder-Mead с штрафом за большие β
- Этап 3: Nelder-Mead без штрафа (чистый mismatch)
- Переменные: k₁, k₂, k₃, k₄, drift_length (опционально k₅)
